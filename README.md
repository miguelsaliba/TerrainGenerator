# Procedural Terrain Generator

## Controls

| Input  | Action               |
|------- | -------------------- |
| ESC    | Switches cursor mode |
| W      | Move forward         |
| A      | Move left            |
| S      | Move backward        |
| D      | Move right           |
| SPACE  | Move up              |
| LCTRL  | Move down            |
